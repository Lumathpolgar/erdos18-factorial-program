# Nova 4 Data

The factorial half-range summary is an exact finite audit of all targets through `n=13`. The detailed `lambda` arrays are regenerated deterministically and identified by SHA-256.

The `data/lattice/` directory contains exact finite transition audits for frozen occupancy models.

The `data/capacity/` directory contains the certified Nova 1 Study A threshold audit through `n=1,000,000`.

The `data/rainbow/` directory contains the exact target-truncated Nova 1 Study B reduced-rainbow computation for every `20<=n<=80`, including the one-row-per-case CSV, source/checksum index, and deterministic full-audit schema.

No finite dataset is an asymptotic theorem.
